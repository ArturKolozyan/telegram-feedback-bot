# Telegram Feedback Bot v2.0 - Linux

## Запуск

1. Настройте `.env` файл (BOT_TOKEN и MANAGER_CHAT_ID)

2. Запустите бота из этой папки:
```bash
./TelegramFeedbackBot
```

3. Бот автоматически создаст папки:
   - `data/` - для хранения данных (пользователи, ответы, настройки)
   - `reports/` - для CSV отчетов

## Важно!

- Всегда запускайте бота из папки `bot_linux/`
- Не перемещайте исполняемый файл в другое место
- Все данные будут храниться в этой папке

## Новые возможности v2.0

1. **Повторные напоминания** - настраиваемые напоминания для тех, кто не ответил
2. **Система выходных и отпусков** - автоматический календарь РФ + управление отпусками
3. **Месячные отчеты** - автоматические отчеты для сотрудников 1-го числа каждого месяца

Подробнее в документации проекта.
